# Arduino Nano

![Arduino Nano](../img/nano.png)

Carte compacte ATmega328P (même cœur que l'Uno), format breadboard.

## Broches

| Broche | Rôle |
|--------|------|
| **0–13** | E/S numériques (~ = PWM) |
| **A0–A7** | Entrées analogiques |
| **5V / 3.3V / VIN** | Alimentations |
| **GND** | Masses |

## Utilisation

- Mêmes capacités que l'Uno, en plus petit.
- A6/A7 : entrées analogiques seules.
