# TFT display (ILI9341)

![TFT display (ILI9341)](../img/ili9341.png)

240×320 color TFT display (SPI). Shows text, images and graphics.

## Pins

| Pin | Role |
|--------|------|
| **VCC / GND** | Power |
| **SCK / MOSI / MISO** | SPI bus |
| **CS** | Chip select |
| **D/C** | Data/Command |
| **RST** | Reset |
| **LED** | Backlight |

## Usage

- SPI bus. Adafruit_ILI9341 / TFT_eSPI libraries.
- The canvas is decoded and drawn in simulation.
