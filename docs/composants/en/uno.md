# Arduino Uno

![Arduino Uno](../img/uno.png)

ATmega328P board: 14 digital I/O (6 PWM), 6 analog inputs, USB.

## Pins

| Pin | Role |
|--------|------|
| **0–13** | Digital I/O (~ = PWM) |
| **A0–A5** | Analog inputs |
| **5V / 3.3V / VIN** | Power |
| **GND** | Grounds |
| **AREF / RESET** | ADC reference / reset |

## Usage

- Full pinout via the **K** button (poster).
- 5 V logic level.
