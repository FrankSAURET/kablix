# Matrix keypad

![Matrix keypad](../img/keypad.png)

4×3 or 4×4 membrane or key keypad: a matrix of keys, rows × columns.

## Pins

| Pin | Role |
|--------|------|
| **L1–L4** | Rows |
| **C1–C4** | Columns |

## Properties

| Property | Role | Default |
|-----------|------|--------|
| `Colonnes` | Number of columns (3/4) | 4 |

## Usage

- Scan the matrix: drive one row, read the columns.
