# Arduino Mega 2560

![Arduino Mega 2560](../img/mega.png)

ATmega2560 board: 54 digital I/O (15 PWM), 16 analog inputs, 4 UARTs.

## Pins

| Pin | Role |
|--------|------|
| **0–53** | Digital I/O |
| **A0–A15** | Analog inputs |
| **5V / 3.3V / VIN** | Power |
| **GND** | Grounds |

## Usage

- For pin-hungry projects.
- Full pinout via the **K** button.
