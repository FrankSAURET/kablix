# NeoPixel ring

![NeoPixel ring](../img/led-ring.png)

Ring of addressable RGB LEDs (WS2812).

## Pins

| Pin | Role |
|--------|------|
| **VCC** | Power (+) |
| **GND** | Ground |
| **DIN** | Data in |
| **DOUT** | Data out |

## Properties

| Property | Role | Default |
|-----------|------|--------|
| `pixels` | Number of LEDs | 16 |

## Usage

- DIN to a digital pin.
- Circular effects (rotation, gauge).
