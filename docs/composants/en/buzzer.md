# Buzzer

![Buzzer](../img/buzzer.png)

Active piezo buzzer: emits a sound as long as a voltage is applied across its terminals.

## Pins

| Pin | Role |
|--------|------|
| **1** | + terminal (positive) |
| **2** | – terminal |

## Usage

- + terminal to an output pin, – terminal to ground.
- `tone()` (Arduino) to generate frequencies.
