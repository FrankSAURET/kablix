# Push button (6 mm)

![Push button (6 mm)](../img/button-6mm.png)

Small 6 mm tactile button, same behavior as the 12 mm button.

## Pins

| Pin | Role |
|--------|------|
| **1.l / 1.r** | First contact |
| **2.l / 2.r** | Second contact |

## Properties

| Property | Role | Default |
|-----------|------|--------|
| `color` | Color | red |
| `key` | Keyboard shortcut | — |

## Usage

- Same as the 12 mm button: `INPUT_PULLUP` + ground.
- Debouncing recommended.
